## 下载 V2Ray
预编译的压缩包：[GitHub地址](https://github.com/v2ray/v2ray-core/releases)

找到对应平台的压缩包，下载解压即可使用。

## Windows 和 Mac OS 安装方式
通过上述方式下载的压缩包，解压之后可看到 v2ray 或 v2ray.exe。直接运行即可。
